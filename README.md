# ConnectIn

A social media networking platform built using MERN stack.

## Features
- User authentication
- Create posts
- Like/comment system
- Profile management
- Resume downloading from profile data

## Installation
1. Clone repo
2. npm install
3. Add .env file
4. npm run dev

## Tech Stack
- React / Next.js
- Node.js
- Express.js
- MongoDB

## Folder Structure
/backend
/connect-in (i.e frontend)
